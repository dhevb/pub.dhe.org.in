# Pilot Evidence Archive

Chronological operational record for the editorial pilot. Preserve weekly reports here permanently — they become the evidence base for v1.1 planning, onboarding, and external credibility (indexing, partners).

**Version baseline:** v1.0.2 · **Platform:** https://pub.dhe.org.in

## How to use

1. Each **Friday**, copy `week-01.md` to `week-NN.md` (or create the next file) and fill it in.
2. Update [PROJECT_STATUS.md](../PROJECT_STATUS.md) with the executive summary.
3. Roll up themes into [PILOT_FEEDBACK_REPORT.md](../PILOT_FEEDBACK_REPORT.md) as patterns emerge.
4. At pilot end, complete [final-summary.md](./final-summary.md) and [PILOT_SUMMARY.md](../PILOT_SUMMARY.md).

## Files

| File | Purpose |
|------|---------|
| `week-01.md` … `week-NN.md` | Weekly operational snapshot |
| `final-summary.md` | Pilot close-out (links to full summary doc) |

## Commit convention

```
docs(pilot): week-01 report
docs(status): update project status
```
